## Description
This demo introduces **DVC (Data Version Control)**, an open-source tool that extends Git workflows to datasets, ML models, and pipelines.  
In the demonstration, we will show how DVC enables reproducibility and collaboration in machine learning projects by:

- Versioning datasets and models alongside code.  
- Building a simple ML pipeline that can be automatically re-executed.  
- Tracking and comparing experiments with different configurations.  
- Demonstrating rollback to previous dataset/model states.  

The live demo will highlight how DVC integrates with Git while solving the unique challenges of ML workflows, such as handling large files and keeping track of experiments.

**Relevance**
**DVC is directly relevant to MLOps**, as it addresses one of the key DevOps challenges in machine learning: ensuring that experiments, datasets, and models are reproducible and shareable across teams.  
By showing how DVC automates experiment tracking and pipeline management, the demo connects core DevOps principles such as versioning, automation, and CI/CD to the ML lifecycle.
