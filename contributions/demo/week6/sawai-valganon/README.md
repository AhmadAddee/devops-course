## Description

This demo will showcase how Microsoft Sentinel, Azure’s cloud-native SIEM solution, can be used to automate threat detection and response within a DevSecOps workflow. We will connect Azure resources (such as VMs and web apps) to Microsoft Sentinel, configure log ingestion, and enable built-in analytic rules to detect suspicious activities like repeated failed logins and privilege escalation attempts. The demo will present live alert generation, incident investigation, and automated remediation using Azure Logic Apps. All steps will be performed and visualized in real time, including simulation of attacks and immediate response actions, emphasizing both technical difficulty and visual aspect.

**Relevance**

Integrating continuous security monitoring is essential in modern DevSecOps practices. This demo highlights how Microsoft Sentinel enhances visibility and accelerates incident response through automation, directly addressing the need for rapid, effective security in cloud-native application pipelines. The approach supports best practices by “shifting security left,” providing immediate security insights and enforcing organizational resilience against evolving threats.
