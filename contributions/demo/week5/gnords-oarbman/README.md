## Description
Using the IaC tool Pulumi, we will demonstrate the concept of component resources. Pulumi Components allows the creation and grouping of multiple resources into a single reusable unit. To demonstrate this we will build a Pulumi component that uploads a static website to AWS S3. We will then use that component twice to create two independent websites. This demonstrates Infrastructure as Code, showing how we can bundle multiple resources into a single reusable unit. The reusable aspect is important because it allows teams to standardize infrastructure and reduce duplication.

**Relevance**
Reusable components makes it possible to enforce consistency, improve reliability and speed in infrastructure management, all important DevOps principles.
