## Description

The [paper](https://dl.acm.org/doi/10.1145/3551349.3559503) was published in the "IEEE/ACM International Conference on Automated Software"

The paper introduces an automated fault injection platform that generates high-quality data for training AI/ML models to improve system resiliency. It combines fault injection with data collection to learn how failures propagate through complex, multi-cloud applications.


**Relevance**

In a time of growing IT complexity and expensive outages, this paper directly addresses the need for stronger system resilience. The platform offers an automated solution for generating realistic training data. This is a big challenge in developing effective AIOps models.
