## Description

We will present Pulumi that is a modern IaC tool that enables developers to manage cloud infrastructure using common programming languages such as Python, TypeScript, etc. The presentation will contain: a short introduction to Iac, what is Pulumi and how it works, the most popular tools and how Pulumi differs from the more established tools. The presentation will also contain a short code example showing how Pulumi can be used.  

**Relevance**

Pulumi is relevant to devops because it presents a tool that aims to solve problems related to IaC, a relevent subject for devops. The presentation aims to highlight relevant IaC challenges and solutions.
