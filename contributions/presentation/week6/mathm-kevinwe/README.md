## Description

This is a presentation about the techniques used in secret management applications like [HashiCorp Vault](https://developer.hashicorp.com/vault) and [openbao](https://openbao.org/) and their use in simplifying and thereby securing secret use in projects by removing some human factors. Things the presentation will cover:
* Dynamic Secrets
* Secret Leasing
* Encryption as a Service
* Secure Secret Storage

We will present benefits and limitations with using applications like these and which problems they solve well, and which they do not. 

**Relevance**

Modern secure systems need a lot of different secrets and without a good system of managing said secrets it is easy for human error (laziness) to introduce security risks. A secret management system helps with mitigating these risks in a way that is easy to integrate in a devops workflow.
